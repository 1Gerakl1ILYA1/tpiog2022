**Можно:**
```c++
#include <iostream>
 
class Data{
private:
  int a_, b_;
public:
  Data(int a, int b) : a_(a), b_(b){}
};
 
int main(){
  const size_t N = 2;
  Data a[N] = {Data(1, 2), Data(3, 4)};
  return 0;
}
```  
**Нет, так как классы лишь условность, которая описывает данные, и во время работы программы она исчезает, из-за чего сослаться на неё  невозможно.**
